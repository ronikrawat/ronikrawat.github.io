Initialize a Repository

Command: git init

Description: Initializes a new Git repository.

Example:

mkdir my-project
cd my-project
git init

Committing Changes

Command: git commit -m "message"

Description: Commits the staged changes with a descriptive message.

Example:

git add .
git commit -m "Initial commit"

Branching

Command: git branch

Description: Lists all branches in the repository.

Example:

git branch

Command: git branch branch_name

Description: Creates a new branch.

Example:

git branch feature-x

Merging

Command: git merge branch_name

Description: Merges the specified branch into the current branch.

Example:

git merge feature-x

Working with Remote Repositories

Command: git remote add remote_name URL

Description: Adds a new remote repository.

Example:

git remote add origin https://github.com/username/repository.git

Command: git remote -v

Description: Displays the URLs of remote repositories.

Example:

git remote -v

Cloning Repositories

Command: git clone URL

Description: Creates a local copy of a remote repository.

Example:

git clone https://github.com/username/repository.git

Pulling Changes

Command: git pull

Description: Fetches and merges changes from the remote repository to your local branch.

Example:

git pull origin main

Pushing Changes

Command: git push

Description: Pushes your local commits to the remote repository.

Example:

git push origin main

Force Pushing Changes

Command: git push --force

Description: Forcefully pushes changes to the remote repository, overwriting any conflicting changes.

Example:

git push --force origin main

Force Pushing with Lease

Command: git push --force-with-lease

Description: A safer alternative to git push --force that only forces the push if the remote branch hasn't been updated.

Example:

git push --force-with-lease origin main

Removing Files, Branches, and Remotes

Removing Files

Command: git rm filename

Description: Removes a file from the working directory and stages the removal.

Example:

git rm myfile.txt
git commit -m "Removed myfile.txt"

Removing Cached Files

Command: git rm --cached filename

Description: Removes a file from the staging area but keeps it in the working directory.

Example:

git rm --cached myfile.txt
git commit -m "Removed myfile.txt from staging but kept it locally"

Removing Branches

Command: git branch -d branch_name

Description: Deletes a local branch that has been merged.

Example:

git branch -d feature-x

Command: git branch -D branch_name

Description: Forces the deletion of a local branch, even if it hasn't been merged.

Example:

git branch -D feature-x

Removing Remote Branches

Command: git push remote_name --delete branch_name

Description: Deletes a branch from the remote repository.

Example:

git push origin --delete feature-x

Removing Remote Repositories

Command: git remote rm remote_name

Description: Removes a remote repository reference from your local repository.

Example:

git remote rm origin
